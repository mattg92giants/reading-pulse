## Overview
A full-stack web app to show the most shared and read content across Twitter, Reddit, and Google Trends, with a mobile-first layout.

## Deployment

### Backend (Flask)
1. Navigate to `server/`
2. Run:
   ```
   pip install -r requirements.txt
   python app.py
   ```
3. Or deploy on [Render](https://render.com)

### Frontend (React)
1. Navigate to `client/`
2. Run:
   ```
   npm install
   npm start
   ```
3. Or deploy on [Vercel](https://vercel.com) / [Netlify](https://netlify.com)
